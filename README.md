# 🌙 New Islamic Year 1448 AH

A beautifully designed and responsive website created to celebrate and welcome the Islamic New Year 1448 AH. This project provides users with an engaging experience through a modern interface, meaningful content, and attractive visuals related to the significance of Muharram and the Islamic New Year.

## ✨ Features

- 🌙 Islamic New Year 1448 AH themed design
- 📱 Fully Responsive Layout
- 🎨 Modern and Clean User Interface
- 🕌 Information about the Islamic New Year
- 📖 Inspirational Islamic Content
- ⚡ Smooth Navigation and User Experience
- 💻 Built with HTML, CSS, and JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🌟 Purpose of the Project

This project was developed to commemorate the beginning of the Islamic New Year (1448 AH) and spread awareness about its significance, history, and values in an interactive and visually appealing way. The website aims to provide visitors with meaningful information while maintaining a modern web experience.
